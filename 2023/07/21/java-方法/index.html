构造方法

  • 无参构造方法，每个类中都会默认存在一个，当你定义了一个构造方法时，这个默认的就会失效

    class Preson(){​

    ​ public Preeson(){}

    }

  • 有参构造方法

    class Preson(参数1，参数2….){

    ​ public Preeson(){}

    }

    构造方法一般用于实例化对象的初始化

构造方法也是可以重载的
